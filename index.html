            발급 방법은 아래 안내 참고. 키는 이 브라우저에만 저장됩니다.
          

        


        

          
          
        


        

          ❓ Google Cloud Vision API 키 발급 방법
          
    
            
  1. Google Cloud Console 접속 → 구글 계정 로그인
    2. 
            
  2. 상단 프로젝트 선택 → 새 프로젝트 (이름 자유)
    3. 
            
  3. 좌측 메뉴 → API 및 서비스 → 라이브러리
    4. 
            
  4. "Vision" 검색 → Cloud Vision API → 사용 설정
    5. 
            
  5. 좌측 메뉴 → API 및 서비스 → 사용자 인증 정보
    6. 
            
  6. + 사용자 인증 정보 만들기 → API 키 클릭
    7. 
            
  7. 생성된 키 복사 → 위 입력란에 붙여넣고 저장
    8. 
            
  8. (권장) 키 옆 ✏️ 편집 → API 제한사항에서 Cloud Vision API만 선택
    9.
