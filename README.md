# TagScanner Web — 옷 태그 OCR (브라우저 버전)

옷 태그를 촬영/선택하면 OCR로 **상품명/브랜드/가격/사이즈/시리얼**을 자동 추출하고
저장·편집·조회가 가능한 **순수 웹 앱**입니다.

- 설치/빌드 도구 없음 — `index.html`만 열면 끝
- **Tesseract.js**로 브라우저에서 직접 OCR (한국어+영어 지원)
- **localStorage**에 저장 (오프라인, 비용 X)
- 모바일 브라우저에서 열면 **카메라 직접 촬영** 가능

---

## 실행 방법

### 방법 ① — `npx serve` (가장 간단, 권장)

cmd에서:

```cmd
cd C:\Users\k3570\Desktop\Claude\TagScanner-Web
npx serve
```

처음 실행할 때 `serve` 패키지 설치 여부 묻으면 **`y`** 입력.

서버가 시작되면 이런 메시지 나옴:
```
Local: http://localhost:3000
```

브라우저에서 `http://localhost:3000` 접속.

### 방법 ② — VS Code Live Server 확장

1. VS Code에서 `TagScanner-Web` 폴더 열기
2. 확장 마켓플레이스에서 **Live Server** 설치
3. `index.html` 우클릭 → **Open with Live Server**

### 방법 ③ — Python (Python 깔려있으면)

```cmd
cd C:\Users\k3570\Desktop\Claude\TagScanner-Web
python -m http.server 3000
```

그리고 브라우저에서 `http://localhost:3000` 접속.

> ⚠️ **`index.html` 파일을 더블클릭하면 안 돼요!** Tesseract.js가 워커/언어 데이터를 CDN에서 불러오는데 `file://`에서는 CORS 막힙니다. 반드시 위 셋 중 하나로 서버 띄워서 접속하세요.

## 사용 흐름

1. **메인**: `📸 카메라로 촬영` 또는 `🖼 갤러리에서 선택`
2. **OCR 자동 인식** — 처음 1회만 한국어 데이터 다운로드 (~10MB, 캐시됨)
3. **5개 필드 자동 채움** + 사용자가 수정 가능
4. **메모** 추가 후 **저장**
5. **📁 저장됨** 탭에서 다시 보기/수정/삭제

## 모바일에서 사용하려면

PC에서 `npx serve` 실행 후, 같은 Wi-Fi에 연결된 폰 브라우저에서:

```
http://<PC_IP>:3000
```

PC IP 확인:
```cmd
ipconfig
```
→ `IPv4 주소` 항목 확인 (예: `192.168.0.10`)

폰 브라우저로 `http://192.168.0.10:3000` 접속하면 카메라 버튼 누를 때 **폰 카메라가 바로 열림** 📱✅

## 데이터 저장 위치

브라우저 `localStorage`에 저장됨. **그 브라우저/프로필에서만** 보입니다.
- 시크릿 모드 닫으면 사라짐
- 다른 브라우저 가면 안 보임
- 브라우저 데이터 지우면 같이 사라짐

용량 한계: 브라우저당 보통 5~10MB. 사진은 자동으로 1280px로 리사이즈/JPEG 압축해서 저장하지만, 수십 장 쌓이면 가득 찰 수 있음.

> 백업/내보내기 기능은 추후 추가 가능 (CSV/JSON 다운로드 등)

## 핵심 동작

### OCR
- `Tesseract.js v5` (CDN)
- 언어 모델: `kor + eng`
- 첫 실행 시 traineddata 자동 다운로드 → 브라우저 캐시
- 완전 클라이언트 사이드, 서버 없음, 비용 없음

### 필드 자동 분류 (`parser.js`)
| 필드 | 인식 규칙 |
|---|---|
| 가격 | `12,000원`, `₩12,000`, `$19.99`, "가격:" 키워드 |
| 사이즈 | `S/M/L/XL/XXL/FREE`, `85~130`, `25~50` |
| 시리얼 | "STYLE/MODEL/품번" 키워드, 또는 영문+숫자 7자 이상 |
| 브랜드 | 위 셋에 안 잡힌 첫 의미있는 라인 |
| 상품명 | 남은 라인 중 가장 긴 것 |

빗나가면 화면에서 직접 수정. 원문은 "OCR 원문 보기"로 확인 가능.

## 트러블슈팅

**OCR이 너무 오래 걸림 / 멈춤**
→ 첫 실행은 한국어 모델 다운로드(~10MB)로 30초~1분 걸려요. 이후엔 5~15초.

**`Tesseract is not defined` 에러**
→ 인터넷 연결 확인 (CDN에서 Tesseract.js 받아옵니다)

**카메라가 안 열림 (PC)**
→ PC는 `<input capture>`가 무시돼서 파일 선택 다이얼로그가 뜸. 정상.
   카메라 직접 쓰고 싶으면 폰 브라우저로 접속 (위 "모바일에서 사용하려면" 참고)

**저장 실패**
→ localStorage 가득 찼을 수 있음. 오래된 항목 삭제하세요.
